# Azmik (Next.js + NestJS)

این نسخه یک **MVP واقعی** از آزمیـک است که هسته‌های اصلی را آماده می‌کند تا بتوانید به‌صورت مرحله‌ای پروژه را به سطح بین‌المللی برسانید.

## معماری
- `frontend/`: رابط کاربری با Next.js + TypeScript
- `backend/`: API با NestJS + TypeScript

## قابلیت‌های پیاده‌سازی شده در این نسخه
- صفحه ورود یکپارچه با انتخاب نقش (مدیر/معلم/دانش‌آموز) + ورود جداگانه ادمین
- تم سفید با رنگ اصلی `#0055ff` و هدر مات
- ساختار داشبورد + منوی کناری
- صفحه چت با الگوی تلگرام، انتخاب چت معلم/ادمین، نمایش کد چت و امکان تایپ پیام
- API ورود واقعی با نام کاربری/رمز و کنترل کد مدرسه/کلاس برای معلم/دانش‌آموز
- API کاربران (مشاهده/ایجاد/تعلیق/حذف)
- API تنظیمات ادمین (بستن چت، ریست کامل)
- API اطلاعیه‌ها
- API آزمون‌ها (ایجاد و لیست)

## اجرای پروژه
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run start:dev
```

## نقشه توسعه بعدی
قابلیت‌های بزرگی مثل آپلود فایل و ویس، استیکر متحرک، تصحیح پیشرفته، کنترل حساب کامل، مدرسه/کلاس چندسطحی، چت‌بات DeepSeek، جستجوی گوگل درون‌اپ، و کلاس آنلاین زنده در فازهای بعدی قابل اضافه شدن هستند.
